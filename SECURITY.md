# Security Policy

## Supported Versions

Only the latest release of the graphical user interface (GUI) is actively supported with security updates. 

| Version | Supported |
| ------- | --------- |
| v1.0.1 (Latest) | :white_check_mark: |
| v1.0.0 (Pre-release) | :x: |
| CLI (Legacy) | :x: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it immediately. 

To report a vulnerability, please contact us directly by reaching out via:
* **GitHub**: [https://github.com/toxicbishop](https://github.com/toxicbishop)

Please do not open a public GitHub issue for security vulnerabilities until they have been addressed.
